# TechEdge Motors - Customer Complaint Form

This is a simple PHP and MySQL project I made as part of my core PHP assessment at TOPS Technologies.

---

## About the Project

TechEdge Motors wanted to digitize their customer complaint process. So I made a basic web form where customers can submit their vehicle complaints. The data gets saved in a MySQL database and can be viewed on a separate page.

---

## Technologies Used

- HTML / CSS
- PHP
- MySQL
- XAMPP (for local server)


## What I Learned

- How to handle HTML forms using PHP
- How to connect PHP with MySQL and run INSERT and SELECT queries
- How to show database records in a table format
- Basic validation and error handling
